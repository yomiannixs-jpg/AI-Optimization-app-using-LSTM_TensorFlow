# NGX Pro Terminal


## Pro Terminal features
- NGX Optimizer: upload multi-sheet XLSX or any CSV/XLSX (wide/long), run LSTM regime classifier (TensorFlow) or rule-based fallback.
- Market Terminal: Twelve Data OHLCV via Time Series endpoint (set TWELVEDATA_API_KEY).
- Strategy Lab: common research analytics.


- Market Terminal includes SMA/EMA, RSI, MACD panels computed locally from OHLCV.
